# Database Module

Production-grade RDS PostgreSQL module with automated secret management, SSM parameter discovery, encryption at-rest, custom parameter tuning, and CloudWatch monitoring.

## Architecture

```
Database Module
├── RDS PostgreSQL 16      (gp3 storage, encrypted, custom params)
├── Secrets Manager        (auto-generated password, JSON credentials)
├── SSM Parameter Store    (endpoint discovery for ECS tasks)
├── CloudWatch Log Group   (PostgreSQL slow query logs)
├── CloudWatch Alarms      (CPU, storage, connections)
└── Enhanced Monitoring    (optional, IAM role + metrics)
```

**Security Pattern:** Password is generated by `random_password`, stored in Secrets Manager as JSON (username, password, host, port, dbname, url). ECS tasks read the secret via Task Execution Role — password never appears in `terraform.tfvars` or environment variables.

## Usage

```hcl
module "database" {
  source = "../../modules/database"

  project_name = "observability"
  environment  = "lab"

  # From network module
  vpc_id          = module.network.vpc_id
  data_subnet_ids = module.network.data_subnet_ids

  # From security module
  data_security_group_id = module.security.data_security_group_id

  # RDS configuration
  instance_class        = "db.t3.micro"
  db_name               = "orders"
  allocated_storage     = 20
  max_allocated_storage = 50

  # Production toggles
  multi_az                = false   # true for production
  backup_retention_period = 1       # 30 for production
  deletion_protection     = false   # true for production
  skip_final_snapshot     = true    # false for production

  # Monitoring
  enhanced_monitoring_interval = 0     # 60 for production
  enable_cloudwatch_alarms     = true

  common_tags = {
    Module = "database"
  }
}
```

## Design Decisions

| Decision | Rationale |
|----------|-----------|
| `random_password` + Secrets Manager | Password never in tfvars/state. ECS reads via IAM at runtime |
| Custom `aws_db_parameter_group` | Tunable: `log_min_duration_statement`, `shared_buffers`, `log_statement` |
| SSM Parameter Store for endpoints | Service discovery without hardcoding. Change endpoint → update SSM → restart task |
| `gp3` storage type | Better price/performance than gp2. Configurable IOPS without io1 cost |
| `storage_encrypted = true` | AWS default KMS key. No extra cost. CIS Benchmark compliance |
| Conditional Enhanced Monitoring | `monitoring_interval = 0` in lab, `60` in prod — avoid unnecessary IAM role |
| Conditional CloudWatch Alarms | Can disable for pure dev/testing environments |
| `create_before_destroy` on parameter group | Prevent downtime during parameter group changes |
